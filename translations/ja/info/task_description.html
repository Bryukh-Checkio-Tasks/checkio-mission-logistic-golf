小包の配達のために、最短距離を探すことは重要です。 街の間の道路が直線の道である道路地図の単純なモデルを見てみましょう。

あなたは道路地図を 距離行列の表現でグラフとして与えられます。 N 個の街では、行列の大きさはNxNです。 行列(2次元配列)は場所の集合としての距離が含まれます。 例えば、i 番目の列とj 番目の行は街iと街jの間の距離を表します。 もし matrix[i][j] == 0 だったら、それらの街の間に道は存在しないということです。 道には向きがないので matrix[i][j] == matrix[j][i] になり、付け加えるなら行列の対角成分は0です。

あなたは0番目 と (N-1)番目の街の間の最短経路を探さなければいけません。 結果は、その道の合計の長さでなくてはいけません。 もし道が存在しないならば0を返します。

例: この地図は以下の行列によって表されて結果は116です。 

(( 0, 80, 58,  0),
 (80,  0, 71, 80),
 (58, 71,  0, 58),
 ( 0, 80, 58,  0))

追加ルール: これは コードゴルフミッション で、あなたの主なゴールはあなたのコードを出来る限り短くすることです。 コードを短くすればより多くのポイントを獲得することができます。 あなたのスコアは動的でコードの長さに直接関連しています。 参考までにスコアは使われた文字の数に基づいています。 350文字が許容される最大文字数でこの時は0点です。 350文字から1文字減るごとに1ポイント獲得することができます。 例えば200文字の長さのコードでは150ポイント獲得することになります。

入力 整数のタプルのタプルとして道路地図が与えられます

出力 街0 から 街(N-1) までの最短経路の長さを整数で出力します

例 

golf(((0, 80, 58, 0), (80, 0, 71, 80), (58, 71, 0, 58), (0, 80, 58, 0))) == 116

どうやって使われるか これは古典的な経路探索問題ですが、この状況ではより創造性とPythonのトリックに頼る必要があります。 Pythonの機構のより深い知識の解説が助けになるかもしれません。

事前条件
1 < len(matrix) ≤ 20
all(len(matrix[0]) == len(row) for row in matrix)
all(not matrix[i][i] for i in range(len(matrix)))
all(all(matrix[i][j] == matrix[j][i] for j in range(len(matrix))) for i in range(len(matrix)))
all(all(0 ≤ el < 10 ** 5 for el in row) for row in matrix)
