For parcel delivery, it is important to find the shortest path. Let's look at simple model of road map, where roads are straight lines between cities.

You are given a road map as a graph in distance matrix representation. For N cities, the matrix has a size of NxN. The matrix (a two-dimensional array) contains the distances as of sets of points. For example: an element in i-th row and j-th column shows a distance between city i and j. If the matrix[i][j] == 0, then a road doesn't exist between these cities. The roads are undirected, so matrix[i][j] == matrix[j][i], and in addition, the matrix has zero main diagonal.

You should find the shortest path between the 0th and (N-1)th cities. The result must be the total length of this path. If a path doesn't exist then return 0.

For example: this map will is represented by the following matrix and results in 116. 

(( 0, 80, 58,  0),
 (80,  0, 71, 80),
 (58, 71,  0, 58),
 ( 0, 80, 58,  0))

Input: A road map as a tuple of tuples with integers.

Output: The length of the shortest path from city 0 to city (N-1) as an integer.

golf(((0, 80, 58, 0), (80, 0, 71, 80), (58, 71, 0, 58), (0, 80, 58, 0))) == 116
