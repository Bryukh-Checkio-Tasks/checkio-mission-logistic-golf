For parcel delivery is important to find the shortest path. Let's look at simple model of road map, where roads are straight lines between cities.

You are given a road map as a graph in distance matrix representation. For N cities the matrix has NxN size. The matrix (two-dimensional array) containing the distances, taken pairwise, of a set of points. For example an element in i-th row and j-th column show a distance between city i and j. If matrix[i][j] == 0, then a road doesn't exist between these cities. The roads are undirected, so matrix[i][j] == matrix[j][i]. And the matrix has zero main diagonal.

You should find the shortest path between 0th and (N-1)th cities. The result must be a total length of this path. If the path doesn't exist then return 0.

For example this map will be represented with the follow matrix and the result is 116. 

(( 0, 80, 58,  0),
 (80,  0, 71, 80),
 (58, 71,  0, 58),
 ( 0, 80, 58,  0))

Additional rules: This is a code golf mission and your main goal is to make your code as short as possible. The shorter your code, the more points you earn. Your score for this mission is dynamic and directly related to the length of your code. For reference, scoring is based on the number of characters used. 250 characters is the maximum allowable and it will earn you zero points. For each character less than 250, you earn 1 point. For example for 200 character long code earns you 50 points.

Input: A road map as a tuple of tuples with integers.

Output: The length of the shortest path from city 0 to city (N-1) as an integer.

How it is used:

Precondition:
1 < len(matrix) ≤ 20
all(len(matrix[0]) == len(row) for row in matrix)
all(not matrix[i][i] for i in range(len(matrix)))
all(all(matrix[i][j] == matrix[j][i] for j in range(len(matrix))) for i in range(len(matrix)))
